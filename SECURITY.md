# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing [INSERT CONTACT EMAIL].

- Do not create public issues for security vulnerabilities.
- Provide as much detail as possible to help us understand and address the issue quickly.
- We will acknowledge your report within 3 business days and provide a plan for resolution.

## Supported Versions

We aim to support the latest major release. Please check the repository for the current supported version.

Thank you for helping keep this project and our community safe!
